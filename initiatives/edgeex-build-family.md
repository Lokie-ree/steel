# Initiative — EdgeEx Build Family

**Vision:** A family of supplemental interactive builds aligned to the EdgeEx Algebra II/III courses — the agile layer on top of the district curriculum, never a replacement. The leverage fact from [[../wiki/edge-ex-courses]]: ~20 lessons overlap between the two courses; build once, serve both preps.

**Current state (2026-07-18):** North star of the Fall Runway sprint. First build scoped-in-name-only: **function transformation explorer** (`F-BF.B.3`, cited by ~10 lessons in both courses — highest reuse in the bank). Sprint rule: brainstorm before code; home decision pending (stack boundary says R3F → creative-lab; if it lands plain JSX/SVG it belongs in course-lab).

**Harvested inputs:** [[../wiki/patterns]] §5 (durable stage machine — any multi-round module must survive a closed Chromebook lid), §6 (resilient sink), §1 (if the family grows a lesson-alignment corpus, it starts as markdown + sync). creative-lab's earned-reveal round structure ([[../archaeology/creative-lab]]) is the pedagogy template.

**First sprint-sized slice:** the transformation-explorer brainstorm session already on the sprint's focus list — output is a scoped plan (one module vs. family seed, home repo, standard mapping), not code. Subsequent slices: one build per sprint, each tagged to its EdgeEx lesson IDs so reuse across both courses is checkable.
